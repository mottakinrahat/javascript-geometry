The Question

1.What is DOM?What is the purpose of Dom?

Ans: DOM or Document Object Model is a programing interface.DOM is used in web documents.DOM treats an HTML document as a tree structure. Purpose of DOM:
Dom can detect the html element with its logical tree.Dom can detect any html element by HTML element's ID,Class name or tag name etc, so that programs can change the document ,style and content.

2.How to select element from DOM?

Ans:DOM can select the elements of HTML using Javascript.In 5 way DOM can select the element:
1.getElementsByTagName()

2.getElementsByClassName()

3.getElementsByID()

4.querySelector()

5.querySelectorAll()

4.What is callback function?

Ans:Callback function is a function passed into the another function.It passed as an argument.There need whether a parameter or not.Callback function make the work very much easier.
